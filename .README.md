dotfiles
========

Here lies the dotfiles of @TooTallNate. I use this repo to sync up my dotfiles on
my various UNIX machines.

To install:

``` bash
cd ~
git clone git://github.com/TooTallNate/dotfiles.git
cd dotfiles
cp -rv `ls -A -1` ~
cd ~
rm -rf dotfiles
```
